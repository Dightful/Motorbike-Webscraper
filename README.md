# Python_projects
Motorbike Webscraper made in 2/6/21 

This program uses a combination of beautiful soup and selenium depending on each website. Allowing to scrape motorcycle companies bikes for sale.

Data scraped includes, make, model and price. 

Companies websites scraped include Aprilia, BMW, Ducati, Honda, KTM, Kawasaki, MV Agusta, Suzuki, Yamaha.

This project originated from being interested in motorcycles and wanting to scrape the data of the bikes to compare them against one another easier.
Made at the age of 15
